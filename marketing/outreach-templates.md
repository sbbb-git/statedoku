# Outreach templates — Statedoku

5 ready-to-customize email templates for backlink + traffic generation.
Each is short (under 120 words), personal, and asks for one concrete thing.

**Rules:**
1. Send 5-10 per week max, not in batches.
2. Always personalize the first sentence — reference something specific the recipient wrote/posted.
3. Never use ALL the brackets verbatim — replace + read aloud before sending.
4. Subject lines under 50 chars.
5. Plain text only — no HTML, no signatures with images.

Expect:
- 30-50% open rate (subject + sender matters)
- 5-15% reply rate (quality of personalization matters)
- 1-5 backlinks total → still high-quality SEO juice

---

## ❌ Anti-patterns to avoid

| Don't | Why |
|---|---|
| Send same mail to 50 people via BCC | Trivial spam detection, hurts sender reputation |
| Use a mail merge tool | Looks templated, lower reply rate |
| Include attachments on first contact | Triggers spam filters |
| Ask for "feedback" without specifics | Vague asks get vague answers (= none) |
| Follow up more than 1 time | Annoying; if no reply after 1 ping, move on |
| Use "Hi guys", "Hey there" | Cold and lazy. Always use first name. |
| Mention "I noticed you have a high DA" | Shows you only care about backlinks. Burns the bridge. |
