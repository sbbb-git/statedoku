Short blurb (50 words)

Statedoku is a daily geography puzzle: a 3×3 grid where each cell must be a US state that satisfies its row clue and its column clue (for example, "borders Canada" × "no income tax"). 100+ rotating clues, free, multilingual, no signup. Play: https://statedoku.com

Medium blurb (130 words)

Statedoku is a free daily geography puzzle, Wordle-style for US states. Every day at midnight a new 3×3 grid drops. Each cell must be a US state that satisfies both its row clue AND its column clue — for example "borders Canada" × "no income tax" (answer: Alaska, Washington, or New Hampshire). 100+ rotating clues across regions, borders, history, economy, and geography. Every puzzle has exactly one solution — no guessing required. What sets it apart from other Wordle clones: logic + geography, not word-guessing. Free, no ads, no signup, no tracking. Multilingual (EN/FR/ES). Built by a solo indie developer. Launched June 1, 2026. Play at https://statedoku.com

⭐ Key features

  • 3×3 grid logic puzzle — each cell must satisfy both row + column constraint
  • 100+ rotating clues across geography, history, economy, borders, regions
  • Provably unique solution — no guessing required
  • Daily puzzle drops at midnight UTC
  • 3 lives per puzzle — forgiving but still challenging
  • Free, no signup, no ads, no tracking
  • Multilingual: English, French, Spanish
  • Privacy-first: Cloudflare Web Analytics only (no GA4, no cookies)
  • Free public JSON API for developers
  • Open educational data with 35+ deep-dive guides on US states
