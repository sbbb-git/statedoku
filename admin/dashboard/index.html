📝 Where to edit content

  • State data → /data/states.json (50 states × 50 properties)
  • Constraint labels → /data/translations.json (EN/FR/ES)
  • Daily puzzle algorithm → /js/puzzle.js
  • Site copy → /index.html, /fr/, /es/, /faq/, /about/, /how-to-play/
  • Design / CSS → /css/style.css
  • Logo → /logo-mark.svg + /favicon.svg
  • Global feature flags → /config.js (ads on/off, admin hash)
  • Admin password → run node bin/set-admin-password.mjs locally
  • Twitter bot → /bot/src/worker.js (deploy with wrangler)
  • SEO sitemap → /sitemap.xml

All edits go via git push → auto-deploy to Cloudflare in ~30s.
