Statedoku 🇺🇸

How to play

The rules

  1. The grid is 3×3. Each row and each column has a constraint label.
  2. Tap a cell, type a few letters of a US state, pick from the list.
  3. The state you place must match both the row and column constraint.
  4. Each state can be used only once. You have 3 mistakes per day.

Example

Row: Pacific coast. Column: Borders Mexico. The cell where they meet → California.
