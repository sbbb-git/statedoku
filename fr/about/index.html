À propos de Statedoku

Le puzzle quotidien des États américains, expliqué

Qu'est-ce que c'est

Statedoku est un puzzle quotidien gratuit dans lequel vous remplissez une grille 3×3 avec des États américains. Chaque ligne et chaque colonne a une contrainte (genre « frontière avec le Mexique » ou « nommé d'après un roi »), et chaque cellule doit contenir un État qui respecte les deux. Chaque État ne peut être utilisé qu'une fois. Vous avez droit à 3 erreurs. Résolvez la grille avant d'épuiser vos chances.

Un nouveau puzzle sort tous les jours à minuit, le même pour tout le monde dans le monde entier. Résolvez-le, partagez votre grille emoji, revenez demain.

Pourquoi on l'a créé

Wordle a prouvé que les puzzles quotidiens simples peuvent être addictifs — il suffit d'une contrainte petite et intelligente, et d'une expérience partagée à l'échelle mondiale. On voulait appliquer la même recette à la géographie : un petit rituel qui vous apprend quelque chose en jouant.

Statedoku récompense la vraie connaissance de la géographie américaine (où sont les États, leurs frontières, leurs spécificités) plutôt que des astuces de vocabulaire. Tous les jours, n'importe qui avec des bases solides peut résoudre la grille ; les jours difficiles creusent l'étymologie des noms, les dates d'admission ou l'histoire politique.

Ce qui le rend différent

Il existe plein de puzzles quotidiens. Statedoku se distingue parce que :

  • Combinatoire, pas aléatoire. Chaque grille a mathématiquement une seule solution. Pas de hasard.
  • ~100 contraintes différentes tournent — régions, frontières, montagnes, ceintures économiques, histoire, politique, origine des noms, sports, motifs alphabétiques — pas deux jours pareils.
  • Mobile first. Pensé pour les pouces. Tapez sur une cellule, tapez 3 lettres, choisissez un État. Bottom-sheet, grosses zones tactiles, safe area iPhone.
  • Multilingue. Anglais, français, espagnol — toutes les contraintes traduites.
  • Privacy-first. Aucun compte, aucun tracking utilisateur, aucune base serveur. Votre progression reste sur votre appareil.

Sous le capot

Statedoku est un site statique hébergé sur Cloudflare Pages. Le générateur de puzzle tourne entièrement dans votre navigateur — à partir de la date du jour, il produit déterministiquement une et une seule grille. Conséquences :

  • Pas de backend à attaquer ou maintenir
  • Le site charge en moins de 100 ms partout dans le monde (CDN Cloudflare)
  • Tous les joueurs d'une même date voient exactement le même puzzle
  • Vous pouvez jouer hors-ligne une fois la page chargée (PWA)

Roadmap

Ce qui arrive, dans l'ordre :

  • Archive — rejouer n'importe quel puzzle passé
  • Mode difficile — contraintes plus obscures, moins d'aides
  • Thèmes quotidiens — lundi géographie, mardi histoire, etc.
  • Premium optionnel — sans pub, archive complète, thèmes, pour le prix d'un café
  • Plus de pays — Eurodoku (UE), Asiadoku (Asie), etc.
