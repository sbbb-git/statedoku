Intégrer Statedoku sur ton site

Insère n'importe lequel de nos mini-jeux de géographie américaine dans ton blog de classe, ton ENT, ton site éducatif ou ton article. Pur iframe — pas d'inscription, pas de frais, aucun pistage supplémentaire de tes visiteurs.

Comment intégrer

  1. Choisis un jeu ci-dessous. Chacun est entièrement jouable en iframe.
  2. Copie le code sous chaque aperçu.
  3. Colle-le partout où le HTML est autorisé (Notion, WordPress, Google Sites, Pronote, Canvas, Squarespace, Substack…).

Règles d'usage

  • Libre pour tout usage éducatif, journalistique ou personnel.
  • Garde le petit lien « Powered by Statedoku.com » visible en bas de l'iframe — c'est ce qui permet à ce service de rester gratuit.
  • Pas besoin de demander — intègre directement. Si tu publies sur un site à fort trafic, écris-nous à contact@statedoku.com pour qu'on puisse te remercier.
