NJ est l'abréviation USPS officielle à 2 lettres pour New Jersey. Cette page lance le quiz Abréviations USPS commençant par NJ — associez le code à l'État.

Pourquoi NJ ?

L'USPS a standardisé les abréviations d'États à 2 lettres en 1963 pour les cartes perforées des premières machines de tri postal. Le code NJ reprend les 1-2 lettres distinctives de New Jersey.
