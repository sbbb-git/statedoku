MA es la abreviatura USPS oficial de 2 letras para Massachusetts. Esta página lanza el quiz Abreviaturas USPS comenzando por MA — empareje el código con el estado.

¿Por qué MA?

USPS estandarizó las abreviaturas de estados a 2 letras en 1963 para tarjetas perforadas de las primeras máquinas clasificadoras de correo. El código MA toma las 1-2 letras distintivas de Massachusetts.
