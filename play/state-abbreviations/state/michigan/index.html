MI is the official USPS 2-letter abbreviation for Michigan. This page launches the USPS Abbreviations quiz starting with MI — match the code to the state.

Why MI?

The USPS standardized state abbreviations to 2 letters in 1963 to fit punched cards in early mail-sorting machines. The code MI takes the first 1 or 2 distinctive letters of Michigan. It is one of the famously confused 8 M-states (MA/MD/ME/MI/MN/MS/MO/MT).
