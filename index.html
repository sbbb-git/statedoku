The daily US states puzzle, free forever

Statedoku is a free daily puzzle that mixes Sudoku's grid logic with American geography. Every day at midnight, a fresh 3×3 grid drops. Each row and column has a constraint (like "borders Mexico" or "named after a king") and each cell must be filled with a US state that satisfies both. You have 3 mistakes allowed.

Played by geography lovers, casual puzzlers, and Wordle fans looking for something new. No signup, no app to install — just open statedoku.com and play.

    How to play

    Fill each cell with a US state that satisfies both the row and column constraints. Each state can only be used once. Find the unique solution!

    Example: If the row says "West Region" and the column says "Borders Mexico", valid answers include Arizona, California, or New Mexico.

    Green ✅ = Correct placement
    Red ❌ = Wrong state for this cell
